## How to Use

1. Open `travel_recommendation.html` in your browser
2. Use the search bar to search for:
   - `beaches` or `beach`
   - `temples` or `temple`
   - `countries` or `country`
3. Click the Search button to see recommendations
4. Click Clear to reset the view
